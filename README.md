# GenAI Node.js TypeScript App

A simple Generative AI backend using:

- Node.js
- TypeScript
- Express
- OpenAI API

## Setup

1. Install dependencies
2. Add your API key in `.env`
3. Start the app

## Commands

- `npm install`
- `npm run dev`

## Endpoint

### POST `/api/ai/chat`

Request body:

```json
{
  "prompt": "Explain Generative AI in simple words"
}
```
